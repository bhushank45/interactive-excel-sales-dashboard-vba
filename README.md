# Interactive Excel Sales Dashboard VBA

## Overview
Interactive Sales Dashboard created using Microsoft Excel and VBA.

## Features
- Pivot Tables
- Pivot Charts
- Slicers
- VBA Automation
- Dynamic Dashboard Filtering
- Sales Performance Analysis

## Tools Used
- Microsoft Excel
- VBA
- Pivot Tables
- Conditional Formatting

## VBA Functionality
The dashboard uses VBA macros to dynamically connect and disconnect slicers from PivotTables using checkbox controls.

## Important
Enable macros while opening the `.xlsm` file.
